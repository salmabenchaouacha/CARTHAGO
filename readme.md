# 🌍 Carthago — AI Travel Planner & Services Marketplace

## 📌 Overview

**Carthago** is a hybrid platform that combines an **AI-powered travel planner** with a **marketplace for local services and experiences**.
The goal is to provide users with a seamless journey from **planning a trip** to **booking activities and discovering local artisans** — all in one place.
