# 🌍 Carthago — AI Travel Planner & Services Marketplace

## 📌 Overview

**Carthago** is a hybrid platform that combines an **AI-powered travel planner** with a **marketplace for local services and experiences**.
The goal is to provide users with a seamless journey from **planning a trip** to **booking activities and discovering local artisans** — all in one place.

---

## 🚀 Key Features

### ✈️ AI Travel Planner

* Personalized travel itineraries generated using AI
* Smart recommendations based on user preferences
* Interactive chat for refining travel plans

### 🛍️ Services Marketplace

* Browse and book local experiences (tours, workshops, activities)
* Discover artisans and cultural products
* Vendor profiles and service listings

### 💳 Booking & Payments

* End-to-end booking system
* Secure payment integration
* Multi-step checkout flow

### 👤 User Management

* Authentication and authorization
* User profiles and preferences
* Booking history

### 📊 Admin Dashboard

* Manage users, vendors, and services
* Monitor bookings and transactions
* Analytics and reporting

---


---

## 🧩 Business Model

Carthago uses a **B2B2C hybrid model**:

* **B2C:** End users plan trips and book services
* **B2B:** Vendors (artisans, guides, service providers) list their offerings
